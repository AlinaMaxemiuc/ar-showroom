Despre proiect

AR Car Showroom este o aplicație WebAR care permite vizualizarea automobilelor în Realitate Augmentată direct din browser.

Tehnologii utilizate

  • HTML5 / CSS3
  • Google Model Viewer
  • Formate 3D: GLB și USDZ
  • Hosting HTTPS – GitHub Pages

Context academic

Proiect realizat integral de către student, în cadrul activităților practice de Realitate Augmentată, fără utilizarea unui backend.
