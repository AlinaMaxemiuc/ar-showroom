Proiectul a fost realizat integral de către student.

Aplicația reprezintă un AR Car Showroom web, care permite
vizualizarea unor modele 3D de automobile în Realitate Augmentată,
direct din browser, fără instalarea unei aplicații dedicate.

Utilizatorul poate selecta diferite modele de mașini și le poate vizualiza
în mediul real folosind camera telefonului, prin tehnologii WebAR.

Tehnologii utilizate:
-> HTML5
-> CSS3
-> Google Model Viewer (WebXR / Apple Quick Look)
-> Formate 3D: GLB (web / Android) și USDZ (iOS)
-> Hosting HTTPS: GitHub Pages

Aplicația nu utilizează backend, fiind o aplicație web statică,
axată pe demonstrarea conceptelor de Realitate Augmentată în mediul web.
